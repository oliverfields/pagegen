Builds .deb and python package (for pip install)

1. Create git tag
   $ git tag -a <tag> -m '<describe tag>'
   $ git push origin <tag>

2. Ensure version correct in files
    - pagegen
    - setup.py

3. Build deb (do first, bacause updates changelog)
   $ mkdir <build dir>
   $ ./build <tag> <build dir>

4. Build python package
   $ python setup.py sdist 


If need to delete remote tag:
$ git tag -d <tag>
$ git push origin :refs/tags/</tag>
